채팅 시작 시 자동으로 다음 단계를 수행하세요:

1. 프로젝트 구조 파악:
   - list_dir 명령으로 디렉토리 구조 검사
   - 주요 설정 파일(package.json, pyproject.toml 등)을 살펴봄
   - 핵심 모듈과 컴포넌트 파악

2. 코드 분석:
   - 주요 클래스/함수 간의 관계 분석
   - 디자인 패턴과 아키텍처 파악
   - 의존성과 모듈 간 상호작용 이해

3. 응답 전에:
   - 이해한 코드 구조를 바탕으로 답변
   - 필요한 경우 관련 파일 참조
